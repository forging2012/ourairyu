买好车前端团队召集小伙伴啦

想迎娶白富美，开豪车通向人生巅峰吗？那就加入我们吧！

某位不愿意透露姓名的名人

相信骨骼精奇、聪明伶俐的你已经心领神会——对！没错！！我们，需要你！！！

「买好车」是一家在行业中数一数二的汽车电商公司，现因业务扩张需要有理想抱负并且身揣两把刷子的你加入，与我们一同实现理想，成为人生赢家！

入职时会为你配备 MacBook Pro 和大屏显示器；有免费的零食和早、晚餐；不定期的团建活动。

技术团队中很多人来自阿里巴巴，拥有开放的氛围。开发部每周都有分享会，让你能够接触了解整个开发环节的方方面面的知识；前端团队内部也会轮流进行分享，帮助你巩固基础及了解流行技术、工具。跟我们在一起，你会与团队一起飞快成长！

前端团队所采用的技术栈主要有：Sass 搭配 Compass 开发页面；使用 Bower 进行第三方库的管理；利用 FIS3 进行构建。更多信息请看团队的 GitHub repo。

为了使我们团队变得更加牛逼，需要你的力量！所以，希望你拥有以下特质：

  1. 拥有技术热情和对技术问题深究的态度
  2. 善于思考总结并乐于分享
  3. 对性能优化和前端工程有自己的理解
  4. 深刻理解「结构、表现、行为相分离」
  5. 能够写出语义化的对搜索引擎友好的 HTML
  6. 能够写出少 hack 且兼容性高的 CSS
  7. 熟悉 JavaScript 并掌握 jQuery
  8. 熟练运用 Sass 与 Compass
  9. 了解 Bower、Gulp、Grunt、FIS 等工具
  10. 具备 PC 端和移动端开发经验

最重要的事情差点忘记说了——

薪资不输于行业水平，但具体多少与你能力成正比。另外，公司的位置是在杭州市余杭区梦想小镇，也就是阿里巴巴西溪园区再往西北一点的地方。

有兴趣的童鞋请将简历、GitHub 和（或）个人网站的链接通过邮箱发送到 oulei@maihaoche.com。We're always waiting for YOU!!! :-D
