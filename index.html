畳（たたみ）

「畳」，即 Tatami，是一个运行在浏览器端的 JavaScript 库，提供了共计 0 个有用的属性和方法使新项目的前端开发变得简单、轻松！方法主要分为数据操作和项目开发两大类。大部分 API 的设计参考了 ECMAScript 5、PHP、jQuery 等语言和库，很容易就能够上手。

本项目托管于 GitHub。如果发现 bug 或有任何建议，欢迎到 issue 页面提出。

运行环境

browser

获取浏览器的相关信息，包括种类及版本号，以 {BROWSER_TYPE: true, version: "BROWSER_VERSION"} 的形式返回。其中 BROWSER_TYPE 的可能值有：

  • msie
  • chrome
  • firefox
  • safari
  • opera

当值为 chrome 或 safari 时，会多出一个 {webkit: true} 属性；当值为 firefox 时，会多出一个 {mozilla: true} 属性。

{% highlight js %} // IE 浏览器 Tatami.browser; => {msie: true, version: "11.0"} // Chrome 浏览器 Tatami.browser; => {chrome: true, version: "34.0.1847.131", webkit: true} {% endhighlight %}
platform

获取平台的相关信息，包括种类、版本号及是否可触摸，以 {PLATFORM_TYPE: true, version: "PLATFORM_VERSION", touchable: true} 的形式返回。其中 PLATFORM_TYPE 的可能值有：

  • windows
  • macintosh
  • iphone
  • ipod
  • ipad
  • ...

当值为 iphone、ipod 或 ipad 时，会多出一个 {ios: true} 属性。

{% highlight js %} // Windows 7 Tatami.platform; => {touchable: false, version: "7", windows: true} // iPhone 5s Tatami.platform; => {touchable: true, version: "7.1.2", iphone: true, ios: true} {% endhighlight %}
