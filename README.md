# Rōnin (Code name: Revolution)

本项目是一个 **DOM 无关的 JavaScript 解决方案、增强库**，旨在弥补原生 JavaScript 的不足，使开发者们在编写 JavaScript 功能模块时少写些代码，更加得心应手！
