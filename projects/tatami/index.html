Tatami(0.2.2)

  • Overview
  • Reference

「畳」，即 Tatami，是一个运行在浏览器端的 JavaScript 库，提供了大量有用的属性和方法使新项目的前端开发变得简单、轻松！方法主要分为数据操作和项目开发两大类。大部分 API 的设计参考了 ECMAScript 5、PHP、jQuery 等语言和库，很容易就能够上手。

本项目托管于 GitHub。如果发现 bug 或有任何建议，欢迎到 issue 页面提出。
