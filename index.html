畳（たたみ）

「畳」，即 Tatami，是一个运行在浏览器端的 JavaScript 库，提供了共计 0 个有用的属性和方法使新项目的前端开发变得简单、轻松！

Tatami 中的方法主要分为数据操作和项目开发两大类。大部分 API 的设计参考了 ECMAScript 5、PHP、jQuery 等语言和库，很容易就能够上手。
